The code is to be run in the following order:
1. take "subset" csv file and put in your local directory
2. open "project_preprocessing_data", change the importing and output directories to your own and run the code,
it will import and preprocess dataset and then export the pre-processed dataset to your directory
Pre-processed dataset is also directly available in "subset_2class_agg" file.
2. open "project_EDA" file, change importing directory to your own and run the code, it will show Exploratory Data Analysis results
3. open "Models" file, change importing directory to your own and run the code, it will show Random Forest, SVM, KNN and XGBoost models results
4. open "CNN Models" file, change importing directory to your own and run the code, it will show CNN models results
